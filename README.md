# Jazz Menu Simulator (CLI & GUI)

This project is a Java-based simulation of a telecom USSD menu system (like *111#) built in two versions:
- Command Line Interface (CLI)
- Graphical User Interface (GUI using Swing)

It demonstrates menu navigation, input validation, and GUI handling in Java.

---

## Files Included

### jazzmenu.java
Console-based application using `Scanner` for input.  
Implements nested menus and user interaction via terminal.

### jazzmenuGUI.java
Graphical version using `JOptionPane`.  
Displays menus in dialog boxes for easier user interaction.

---

## How to Run

### Compile the code:
```bash
javac jazzmenu.java
javac jazzmenuGUI.java
````

### Run CLI Version:

```bash
java jazzmenu
```

### Run GUI Version:

```bash
java jazzmenuGUI
```

---

## Features

* Main menu and sub-menus
* Input validation
* Loop-based menu system
* GUI dialog navigation
* Clear modular method structure
* Back navigation support
